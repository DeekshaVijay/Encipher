JPEG
JPEG (Joint Photographic Experts Group) is one of the most widely used image formats, especially for photographs. It uses lossy compression, which means the file size is reduced by removing some image data, resulting in smaller files that load faster. JPEG is ideal for pictures with many colors and gradients, such as natural photos, but it is not suitable for images that need high clarity or transparency
PNG
PNG (Portable Network Graphics) is a high-quality image format that uses lossless compression, meaning it keeps all the original image details without losing quality. PNG supports transparency, making it ideal for logos, icons, screenshots, and images that require a clear background. It handles sharp edges and text very well, but PNG files are usually larger in size compared to JPEG.
SVG
SVG (Scalable Vector Graphics) is a vector-based image format that uses mathematical shapes like lines and curves instead of pixels. This allows SVG images to be scaled to any size without losing quality, making them perfect for logos, icons, illustrations, and graphics used on websites. SVG files are lightweight, easy to edit, and can even be animated or styled using CSS.

DIV
The DIV tag is a block-level container used to group multiple HTML elements together. It helps structure a webpage into sections such as headers, footers, sidebars, and content areas. Designers use DIV to apply CSS styling, layout designs, background colors, borders, spacing, and to organize content in a clean and manageable way. It acts like a box that holds other elements.
SPAN
The SPAN tag is an inline container used to style or highlight small portions of text or content without breaking the flow of the line. Unlike DIV, it does not start on a new line. UI designers use SPAN to color specific words, change font size, or apply CSS to small text portions. It is mainly used for text formatting and applying inline-level styles.
HEADER
The HEADER tag represents the top section of a webpage or a section of content. It usually contains important introductory elements such as the website logo, navigation menu, title, or heading. It helps users quickly identify the purpose of the page. Designers use HEADER to maintain clean layout structure and to improve accessibility and SEO by clearly defining the beginning of a webpage section.
