Glossary (5 Important Terms)

HTML
The basic language used to create the structure of webpages.
CSS
A style language used to design and decorate webpages.
JavaScript
A scripting language used to add interactivity to websites.
Frontend
The visible part of a website that users interact with.
Backend
The server-side part that handles data and logic.
